# Project Statement - Personal Expense Tracker

## Problem Statement

Many people struggle to track their daily expenses and don't know where their money goes. They need a simple way to record expenses and see spending patterns without using complex financial software or mobile apps. This project provides a straightforward command-line tool to log expenses and understand spending habits.

## Scope of the Project

### What's Included
- Add expenses with date, category, amount, and description
- View all recorded expenses
- Search expenses by category
- Calculate total expenses
- See category-wise spending summary
- Delete the last expense entry
- Local file storage (no internet required)

### What's Not Included
- No graphical interface
- No multiple user support
- No budget alerts
- No data export to Excel/CSV
- Cannot edit old expenses
- Cannot delete specific expenses (only the last one)

## Target Users

**Who will use this?**
- Students managing pocket money
- Young professionals tracking daily expenses
- Anyone wanting a simple expense tracker
- People who prefer privacy with local data storage
- Users comfortable with basic command-line tools

## High-Level Features

### 1. Add Expense
Record new expenses with date, category (Food/Transport/Bills/Shopping/Other), amount, and description.

### 2. View All Expenses
See all your expenses in a neat table format.

### 3. Search by Category
Filter and view expenses from a specific category like Food or Transport.

### 4. Calculate Total
Get the total amount you've spent across all expenses.

### 5. Category-wise Summary
See how much you spent in each category with percentages.

### 6. Delete Last Expense
Remove the most recent entry if you made a mistake.

### 7. Simple Menu Interface
Easy-to-use menu with numbered options - just type 1-7 to navigate.
